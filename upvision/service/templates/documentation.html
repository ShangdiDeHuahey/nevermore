Документация

Данная API-инструкция описывает доступные методы API, использующие технологию компьютерного зрения для распознавания лиц, выражений на лицах, направлений взгляда и уровня активности студентов для определения заинтересованности.

Подробнее

Доступные методы API

  • detectFace(image)

    Обнаруживает лица на изображении.



    Пример использования:

    POST /api/detectFace
    Content-Type: image/jpeg



    Выходные данные:

    Status: 200 OK
Content-Type: application/json
            {
  "faces": [
    {
      "id": 1,
      "boundingBox": {
        "x": 100,
        "y": 200,
        "width": 150,
        "height": 150
      }
    },
    {
      "id": 2,
      "boundingBox": {
        "x": 300,
        "y": 150,
        "width": 120,
        "height": 120
      }
    }
  ]
}

  • detectExpression(faceImage)

    Распознает выражение на лице.



    Пример использования:

    POST /api/detectExpression
    Content-Type: image/jpeg



    Выходные данные:

    Status: 200 OK
Content-Type: application/json
{
  "expression": "Улыбка"
}

  • detectGazeDirection(faceImage)

    Определяет направление взгляда человека.



    Пример использования:

    POST /api/detectGazeDirection
    Content-Type: image/jpeg



    Выходные данные:

    Status: 200 OK
Content-Type: application/json

{
  "gazeDirection": "Прямо"
}
